EnsembleKalmanProcesses.EnsembleKalmanProcessModule.EnsembleKalmanProcess — Type
EnsembleKalmanProcess{FT<:AbstractFloat, IT<:Int}

Structure that is used in Ensemble Kalman processes

#Fields

  • u

    Array of stores for parameters (u), each of size [Npar × Nens]

  • obs_mean

    vector of the observed vector size [N_obs]

  • obs_noise_cov

    covariance matrix of the observational noise, of size [Nobs × Nobs]

  • N_ens

    ensemble size

  • g

    Array of stores for forward model outputs, each of size [Nobs × Nens]

  • err

    vector of errors

  • Δt

    vector of timesteps used in each EK iteration

  • process

    the particular EK process (Inversion or Sampler or Unscented)
