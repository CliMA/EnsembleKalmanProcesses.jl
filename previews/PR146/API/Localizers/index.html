EnsembleKalmanProcesses.Localizers.RBF — Type
RBF{FT <: Real} <: LocalizationMethod

Radial basis function localization method. Covariance terms are damped as d(i,j)= |i-j|/l increases, following a Gaussian.

EnsembleKalmanProcesses.Localizers.BernoulliDropout — Type
BernoulliDropout{FT <: Real} <: LocalizationMethod

Localization method that drops cross-covariance terms with probability 1-p, retaining a Hermitian structure.
